Idle

A super simple CSS library.

Idle is designed to be a simple CSS library that you can drop in to start with some alright looks.

Idle is not layout-focused like many other libraries and if that's something you are looking for then you can go use one of the many available. Another great option is to use Idle for apperance and write your own CSS for laying things out.

Check out what it offers and take it for a spin!
